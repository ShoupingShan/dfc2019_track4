## Utilility Functions for 3D Point Cloud Deep Learning

## visualization tool
```shell
sh compile_render_balls_so.sh
python show3d_balls.py
```
### Tips

  **keyboard shortcut:**

  |**Shortcut**| **Function**|
  |:--:|:--|
  |`g`|Show Ground Truth|
  |`p`|Show Prediction Labels|
  |`r`|Show Residual|
  |`n`|Zoom in|
  |`m`|Zoom out|
  |`r`|Reset|
  |`s`|Save image|
  |`q`|Exit|
